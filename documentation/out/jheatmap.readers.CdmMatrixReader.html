Class: CdmMatrixReader

jheatmap.readers.CdmMatrixReader

new CdmMatrixReader

A text separated value file matrix reader. The file has to follow this format: 


         col1    col2
  row1   0.11    0.12
  row2   0.21    0.22

Parameters:

p.url string File url
p.separator string <optional>
